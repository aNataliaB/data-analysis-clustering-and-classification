# Academic Data Analysis Projects
A curated collection of academic, data analysis, and software engineering projects developed during my studies. This repository serves as a portfolio demonstrating practical applications of machine learning, statistical analysis, and data engineering.

## Repository Content
*   **`wielokryterialna-ocena-jakosci-powietrza`**
    *   *Type:* Clustering / Unsupervised Learning
    *   *Description:* Multi-criteria air quality assessment and atmospheric state segmentation using K-Means and Ward's hierarchical clustering.
    *   *Tech Stack:* Python, Cluster Analysis, PCA, UCI Data Repository.

*   **`przewidywanie-rezygnacji-ze-studiow`**
    *   *Type:* Classification / Machine Learning (Supervised)
    *   *Description:* Predicting student dropout and academic success using an end-to-end Python pipeline with multiple classification algorithms and ensemble methods.
    *   *Tech Stack:* Python, Scikit-learn, Pandas, Seaborn.
